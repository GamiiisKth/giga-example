***Class ***
is a final class in java.lang package which extends Object class.
Instance of this class represents classes and interfaces in a running Java application.
It is used to analyze and change dynamic behaviour of a class at runtime.


Some Important Methods of java.lang.Class class

This class defines several methods using which we can get information about
methods, constructors, modifiers and members
of a class at runtime.
